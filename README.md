## サービス概要
> どんなサービスなのかを３行で説明してください。

あなたの基本的な1週間の予定を入力しよう

勉強・仕事だけでなく、部屋掃除・料理作りといった家事の予定も登録しよう

予定の時間になるとアプリが本領発揮。だらけから脱出してやることをやろう

## このサービスへの思い・作りたい理由
> このサービスの題材となるものに関してのエピソードがあれば詳しく教えてください。
> このサービスを思いつくにあたって元となる思いがあれば詳しく教えてください。

RUNTEQでの勉強生活前に、私はなんとしても卒業したいと思っていましたが、そこでネックになりそうな勉強時間1000時間を確保できるように、勉強生活中の1週間の学習スケジュールを組んでおりました。

ですが、最初のうちはスケジュール通りにできても、4か月程経った頃には予定通りにいかなくなっていきました。主に睡眠開始時間が遅くなった分睡眠時間がずれ込んだり長くなったり、休憩中に気になるコンテンツを見つけてしまい休憩時間が長くなったり、勉強のモチベーションが一度休憩を取ると無くなってしまうなどが起こるようになりました。

そのため、上記の(致し方ない用事を除いた)スケジュール外なことが起こっても、スケジュール通りに修正できる機会を与えてくれるアプリがあればいいなと思い、制作しようと思いました。

さらに、学習だけでなく日常生活でも、スケジュール通りに修正できる機会を与えてくれるアプリであれば面白そうだなと思いました。

## ユーザー層について
> 決めたユーザー層についてどうしてその層を対象にしたのかそれぞれ理由を教えてください。

1. 大事な期間中に周辺の誘惑についつい乗っちゃう人
2. 日々やらなきゃいけないこと(家事とか)をサボりがちであることを直したい人

1.は私のようについつい周りの誘惑に負けても、抜け出して頑張ってほしいですね。
今回、既存のアプリと差別できないかと思い、2.もターゲットにしてみました。ただの学習スケジュールの管理アプリよりも汎用性を高くしたいです。

## サービスの利用イメージ
> ユーザーがこのサービスをどのように利用できて、それによってどんな価値を得られるかを簡単に説明してください。

大事な時期には、大体の人は周辺の誘惑に負けないように排除しようとします。スマホを遠くに置いたり、娯楽を部屋から排除したりなどですね。しかし、娯楽に関してはともかく、スマホを遠くに置くというのは、調べたいことをすぐに調べられなくなるため、やや難しいと思います。そうすると、スマホで検索中に別件で気になるコンテンツを見つけてしまった場合、調べものに必要以上の時間を割いてしまうでしょう。これはよく起こりうる現象ですね。このアプリを使えば、そういったスケジュール外なことが起こっても脱出でき、貴重な時間を(無駄に)奪われることを抑えることができるでしょう。

## 機能候補
> 現状作ろうと思っている機能、案段階の機能をしっかりと固まっていなくても構わないのでMVPリリース時に作っていたいもの、本リリースまでに作っていたいものをそれぞれ分けて教えてください。

### MVPリリースまで

・ユーザー登録機能
・スケジュール登録・変更・削除機能(30分単位。30分越えの予定は1つとして繋げられるようにしたい)
・アラート通知機能(音あり。予定の時間になったタイミングだけでなく、適度に通知できるようにしたい)

### 本リリースまで
・スケジュール登録・変更・削除機能改善(スケジュールアプリらしく、簡単な操作で登録できるように改善)
・アラート通知機能改善(アラート通知機能を付けたい予定を決められるように改善。人によっては仕事時間など、アラートが来てほしくないタイミングがあるはず)
・登録した1週間のスケジュールの複数保存機能(登録したいスケジュールが2種類以上あり、都度変更したい人向け。ただしややマニアックなため、この機能でUIが悪くなるようであれば実装しない)
