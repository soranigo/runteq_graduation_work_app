## サービス概要

あなたの基本的な1週間の予定を入力しよう

勉強・仕事だけでなく、部屋掃除・料理作りといった家事の予定も登録しよう

予定の時間になるとアプリが本領発揮。だらけから脱出してやることをやろう

## このサービスへの思い・作りたい理由

RUNTEQでの勉強生活前に、私はなんとしても卒業したいと思っていましたが、そこでネックになりそうな勉強時間1000時間を確保できるように、勉強生活中の1週間の学習スケジュールを組んでおりました。

ですが、最初のうちはスケジュール通りにできても、4か月程経った頃には予定通りにいかなくなっていきました。主に睡眠開始時間が遅くなった分睡眠時間がずれ込んだり長くなったり、休憩中に気になるコンテンツを見つけてしまい休憩時間が長くなったり、勉強のモチベーションが一度休憩を取ると無くなってしまうなどが起こるようになりました。

そのため、上記の(致し方ない用事を除いた)スケジュール外なことが起こっても、スケジュール通りに修正できる機会を与えてくれるアプリがあればいいなと思い、制作しようと思いました。(ただし、モチベーションを上げる機能ではございません。あくまでサポートするアプリです。)

さらに、学習だけでなく日常生活でも、スケジュール通りに修正できる機会を与えてくれるアプリであれば面白そうだなと思いました。

## ユーザー層について

1. 大事な期間中に周辺の誘惑についつい乗っちゃう人
2. 日々やらなきゃいけないこと(家事とか)をサボりがちであることを直したい人

1.は私のようについつい周りの誘惑に負けても、抜け出して頑張ってほしいですね。
今回、既存のアプリと差別できないかと思い、2.もターゲットにしてみました。ただの学習スケジュールの管理アプリよりも汎用性を高くしたいです。

## サービスの利用イメージ

大事な時期には、大体の人は周辺の誘惑に負けないように排除しようとします。スマホを遠くに置いたり、娯楽を部屋から排除したりなどですね。しかし、娯楽に関してはともかく、スマホを遠くに置くというのは、調べたいことをすぐに調べられなくなるため、やや難しいと思います。そうすると、スマホで検索中に別件で気になるコンテンツを見つけてしまった場合、調べものに必要以上の時間を割いてしまうでしょう。これはよく起こりうる現象ですね。このアプリを使えば、そういったスケジュール外なことに対して、アラート通知機能により脱出でき、貴重な時間を(無駄に)奪われることを抑えることができるでしょう。

## サービスの差別化ポイント・推しポイント

スケジュール外なことが起きても、脱出する機会を与えます。このアプリにはさすがに強制的に人を動かすことはできませんが、機会を与えられれば以外と人は動きますよ。その手助けを行います。

具体的には、通知してほしい予定(勉強や家事など)には、予定時刻だけでなく、予定時刻から3分後、10分後、20分後……等、たびたび音出しアラートをするようにします。アラートに慣れがなるべく起きないように、メッセージの内容を複数用意してランダムに表示したり、音声ファイルをランダムに再生するなどの工夫もしております。
なお、通知してほしくない予定(仕事や来客など)には、通知しないように設定できます。

スケジュールアプリやリマインダーアプリには、「googleカレンダー」や「マイルーティン」などがあり、特に「googleカレンダー」については1つの予定に通知を複数追加することができますね。ただし従来のカレンダーアプリと比べると、(人によって使い方が異なりますが)重要な予定を入れることよりも、毎週決まった予定を入れることを想定しております。また、従来のリマインダーアプリと比べると、だらけ防止として予定時刻を過ぎてから複数のアラート通知を行うアプリはあまり見ないのではないでしょうか。

## 本リリースに向けての追加機能

私の考えたアプリ案は、スケジュール通りに実行していても通知する仕様であり、人によっては集中が途切れてしまうというデメリットがあります。ただ、逆に(私のように)集中しているときに通知が来ても気にならないという人もいるため、上記の機能は実装したいと思います。そのうえで、本リリースに向けてこのデメリットを改善するために、集中状態中にもアラートが来る機能だけでなく、集中状態中にはアラートが来ない機能も実装、さらに通知タイミングの別パターン(例:ポモドーロタイマーのように活動開始してから25分後と休憩5分後の通知、予定終了時刻n分前に通知など)を用意し、ユーザーの好きなように切り替えられるようにします。

## 機能候補

### MVPリリースまで

・ユーザー登録機能
・スケジュール登録・変更・削除機能(30分単位。30分越えの予定は1つとして繋げられるようにしたい)
・アラート通知機能(音あり。予定時刻と予定時刻から3分後、10分後、20分後……のタイミングで通知する)

### 本リリースまで
・スケジュール登録・変更・削除機能改善(スケジュールアプリらしく、簡単な操作で登録できるように改善)
・アラート通知機能改善1(アラート通知機能を付けたい予定を決められるように改善。人によっては仕事時間など、アラートが来てほしくないタイミングがあるはず)
・アラート通知機能改善2(予定時刻以降のアラート通知をオフにする機能を実装。集中を切らしたくない人向けその1。さらにアラート通知のON、OFFを切り替えられるようにする)
・アラート通知機能改善3(通知タイミングの別パターンを実装。集中を切らしたくない人向けその2。さらに通知タイミングのパターンを切り替えられるようにする)
・登録した1週間のスケジュールの複数保存機能(登録したいスケジュールが2種類以上あり、都度変更したい人向け。ただしややマニアックなため、この機能でUIが悪くなるようであれば実装しない)

## 使用する技術スタック

* 使用するフレームワーク：Ruby on Rails 7.1
* データベース：MySQL
* デプロイ先：Render
* 使用予定のライブラリ：具体的ではないのですが、音を鳴らす機能があれば使用したい(audio_tagでもよいが、できればexcelのbeepのように、音声ファイルを作らなくとも音を出せる機能が欲しい)
